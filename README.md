# ChiliFlow

ChiliFlow is an Angular application for organizing and tracking cooking flows in a clear, responsive interface. The project focuses on a modern frontend experience built with Angular, using maintainable structure and up-to-date development practices.

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Open `http://localhost:4200/` in your browser.

## Scripts

